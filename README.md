# sales-analytics-dashboard
Interactive Sales Analytics Dashboard built in Power BI using DAX and Power Query to analyze $13.7M sales data with dynamic KPIs, regional insights, and product performance analysis.
# Sales Analytics Dashboard (Power BI)

## Overview
Interactive dashboard analyzing sales performance, profit margins, and product trends.

## Key Metrics
- Total Sales: $13.7M
- Profit Margin: $5.6M

## Features
- Dynamic filters (Year, Country, Product Line)
- Top 5 & Bottom 5 product analysis
- Region-wise performance map
- DAX-based KPI calculations

## Tools Used
Power BI Desktop, DAX, Power Query
